Funcionalidade: Realizar o login#

Como usuário com perfil (1. Diretor, 2. Gerente, 3. Coordenador, 4. Supervisor, 5. Financeiro) no SisGeop, eu quero acessar o sistema para realizar minhas atividades diárias.

Protótipo#

Especificação#

1. LOGOMARCA#

Na tela de login, o sistema irá exibir a logomarca da JTEL para caracterização do ambiente.

2. FORMULÁRIO DE LOGIN#

O formulário de login será composto por apenas dois campos:

Campo Descrição
Email Email do usuário será utilizado como login do usuário no sistema
Senha Senha de acesso do usuário ao sistema

3. ACESSAR O SISTEMA#

QUANDO o usuário clicar no botão Acessar

ENTÃO o sistema valida o formulário de login

3.1 Formulário inválido: Email não informado#

CASO o usuário não tenha informado o Email de acesso

ENTÃO o sistema exibe a mensagem "Email é obrigatório"

3.2 Formulário inválido: Email inválido#

CASO o usuário tenha informado um endereço de email inválido

ENTÃO o sistema exibe a mensagem "Email inválido"

3.3 Formulário inválido: Senha não informada#

CASO o usuário tenha informado a senha de acesso

ENTÃO o sistema exibe a mensagem "O campo senha precisa ser preenchido"

3.4 Dados incorretos#

CASO o usuário tenha informado dados de acesso corretamente

E o sistema não consiga localizar o usuário no banco de dados

ENTÃO o sistema exibe a mensagem "Usuário não localizado!"

3.4 Login bem sucedido#

CASO o usuário tenha informado os dados de acesso corretamente

E o sistema localize o usuário no banco de dados

ENTÃO o sistema exibe a mensagem "Login efetuado com sucesso"

E redireciona o usuário para a área protegida do sistema

4. RECUPERAÇÃO DE SENHA#

QUANDO o usuário clicar no botão Esqueci minha senha

ENTÃO o sistema redireciona o usuário para a tela de recuperação de senha
