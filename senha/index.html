Funcionalidade: Recuperar senha#

Como usuário com perfil (1. Diretor, 2. Gerente, 3. Coordenador, 4. Supervisor, 5. Financeiro) no SisGeop, eu quero recuperar minha senha, quando, por algum motivo, eu me esquecer dela.

Protótipo#

Especificação#

1. LOGOMARCA#

Na tela de recuperação de senha, o sistema irá exibir a logomarca da JTEL para caracterização do ambiente.

2. FORMULÁRIO DE LOGIN#

O formulário de recuperação de senha será composto por apenas um campo:

Campo Descrição
Email Email do usuário utilizado para efetuar login no SisGeop

3. RECUPERAR SENHA#

QUANDO o usuário clicar no botão Recuperar senha

ENTÃO o sistema valida o formulário de recuperação de senha

3.1 Formulário inválido: Email não informado#

CASO o usuário não tenha informado o Email de acesso

ENTÃO o sistema exibe a mensagem "Email é obrigatório"

3.2 Formulário inválido: Email inválido#

CASO o usuário tenha informado um endereço de email inválido

ENTÃO o sistema exibe a mensagem "Email inválido"

3.4 Email não localizado#

CASO o usuário tenha informado o email no formulário de recuperação de senha corretamente

E o sistema não consiga localizar o usuário pelo seu email no banco de dados

ENTÃO o sistema exibe a mensagem "Usuário não localizado!"

3.4 Recuperação de senha bem sucedida#

CASO o usuário tenha informado os dados de recuperação de senha corretamente

E o sistema localize o usuário no banco de dados

ENTÃO o sistema envia um email para o usuário informando a sua senha cadastrada no SisGeop

E exibe a mensagem "Senha enviada com sucesso para o seu email"

E redireciona o usuário para a tela de login

4. VOLTAR PARA O LOGIN#

QUANDO o usuário clicar no botão Voltar para o login

ENTÃO o sistema redireciona o usuário para a tela de login do SisGeop
